Engineering

Our Robot

Engineered for precision, built for excellence

Intake System

Designed to rapidly and reliably collect game artifacts using a bungee-based suspension system. Built with chain-driven power transmission and 312 RPM motors.

Turret Shooter

A fully redesigned 360° turret-based shooter built for accuracy and rapid scoring from multiple field locations. Uses Limelight 3A auto-targeting and dual Rhino flywheels.

Transfer System

A multi-stage wheel conveyor system designed to balance speed, control, and consistency. Uses 96mm Hogback wheels for acceleration.
